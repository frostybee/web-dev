## How to define a variable?

In PHP, you can define variables with various data types:

```php
$name = "John";
$age = 25;
$isStudent = true;
$pi = 3.14;
```

## How to use a variable?   

```php
echo $name;
echo $age;
echo $isStudent;
echo $pi;
```
