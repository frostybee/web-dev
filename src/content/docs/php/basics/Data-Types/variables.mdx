:::note[PHP Variables]
- Variables in PHP are used to store data that can be used throughout your script(s). 
- PHP is a **dynamically typed** language, meaning you don't need to declare variable types explicitly ==> PHP automatically **determines** the type based on the value assigned.
:::

## Variable Basics

### Variable Declaration
PHP variables start with the `$` symbol followed by the variable name:

```php
$name = "John Doe";
$age = 25;
$isStudent = true;
$salary = 50000.50;
```

### Variable Naming Rules
- Must start with `$` followed by a letter or underscore
- Can contain letters, numbers, and underscores
- Case-sensitive (`$name` and `$Name` are different)
- Cannot start with a number

```php
// Valid variable names
$userName = "john";
$user_name = "john";
$_userName = "john";
$userName123 = "john";

// Invalid variable names
// $123user = "john";     // Cannot start with number
// $user-name = "john";   // Hyphens not allowed
// $user name = "john";   // Spaces not allowed
```

## Variable Scope

### A. Global Scope
Variables declared outside functions have global scope:

```php
$globalVar = "I'm global";

function testGlobal() {
    global $globalVar;  // Access global variable
    echo $globalVar;
}

// Alternative using $GLOBALS
function testGlobals() {
    echo $GLOBALS['globalVar'];
}
```

### B. Local Scope
Variables declared inside functions are local:

```php
function testLocal() {
    $localVar = "I'm local";
    echo $localVar;  // Works fine
}

testLocal();
// echo $localVar;  // Error: Undefined variable
```

### C. Static Variables
Retain their value between function calls:

```php
function counter() {
    static $count = 0;
    $count++;
    echo "Count: $count\n";
}

counter(); // Count: 1
counter(); // Count: 2
counter(); // Count: 3
```

### D. Superglobal Variables
Built-in variables available everywhere:

```php
// $_GET, $_POST, $_SESSION, $_COOKIE, $_SERVER, $_ENV, $_FILES
echo $_SERVER['HTTP_HOST'];    // Current domain
echo $_GET['param'];           // URL parameter
echo $_POST['field'];          // Form field

// $GLOBALS contains all global variables
$GLOBALS['customVar'] = "Available everywhere";
```

## Variable References

### A. Reference Assignment
```php
$original = "Hello";
$reference = &$original;  // $reference points to $original

$reference = "World";
echo $original;  // Outputs: "World"

// Both variables point to the same memory location
unset($reference);
echo $original;  // Still outputs: "World"
```

### B. Passing by Reference
```php
function modifyValue(&$value) {
    $value = $value * 2;
}

$number = 10;
modifyValue($number);
echo $number;  // Outputs: 20
```

### C. References with Arrays
```php
$array = [1, 2, 3, 4, 5];

// Modify array elements by reference
foreach ($array as &$value) {
    $value *= 2;
}
unset($value); // Always unset reference after foreach

print_r($array); // [2, 4, 6, 8, 10]
```

## Variable Variables

Use the value of one variable as the name of another:

```php
$var = "name";
$$var = "John";  // Creates $name = "John"

echo $name;      // Outputs: "John"
echo ${$var};    // Also outputs: "John"

// Complex variable variables
$prefix = "user_";
$id = "123";
${$prefix . $id} = "John Doe";
echo $user_123;  // Outputs: "John Doe"
```

### Practical Example
```php
// Dynamic property access
$properties = ['name', 'email', 'age'];
$user = new stdClass();

foreach ($properties as $prop) {
    $user->$prop = "Default value for $prop";
}

echo $user->name;  // "Default value for name"
```

## Constants vs Variables

### Defining Constants
```php
// Using define()
define('SITE_NAME', 'My Website');
define('MAX_USERS', 100);

// Using const keyword (PHP 5.3+)
const DB_HOST = 'localhost';
const DB_PORT = 3306;

// Class constants
class Config {
    const VERSION = '1.0.0';
    const DEBUG = true;
}

echo Config::VERSION; // "1.0.0"
```

### Magic Constants
```php
echo __FILE__;      // Current file path
echo __LINE__;      // Current line number  
echo __FUNCTION__;  // Current function name
echo __CLASS__;     // Current class name
echo __METHOD__;    // Current method name
```

## Variable Best Practices

### Naming Conventions
```php
// Use descriptive names
$userName = "john_doe";           // Good
$u = "john_doe";                  // Bad

// Use camelCase or snake_case consistently
$firstName = "John";              // camelCase
$first_name = "John";             // snake_case

// Boolean variables should be descriptive
$isActive = true;                 // Good
$active = true;                   // Less clear

// Constants in UPPER_CASE
const MAX_RETRY_ATTEMPTS = 3;
```

### Type Safety
```php
// Use strict types (PHP 7+)
declare(strict_types=1);

function addNumbers(int $a, int $b): int {
    return $a + $b;
}

// Input validation
function processAge($age) {
    if (!is_numeric($age) || $age < 0 || $age > 150) {
        throw new InvalidArgumentException('Invalid age provided');
    }
    return (int)$age;
}
```

### Variable Initialization
```php
// Initialize variables before use
$total = 0;
$items = [];
$user = null;

// Use null coalescing operator (PHP 7+)
$username = $_GET['user'] ?? 'guest';
$config = $userConfig ?? $defaultConfig ?? [];

// Null coalescing assignment (PHP 7.4+)
$cache ??= expensive_operation();
```

### Memory Management
```php
// Unset large variables when done
$largeArray = range(1, 1000000);
// ... process array
unset($largeArray);

// Use references for large data structures
function processLargeArray(array &$data) {
    // Process without copying the array
    foreach ($data as &$item) {
        $item = strtoupper($item);
    }
}
```
